子供用の商品販売マーケットプレース

クラインアント

日本企業

概要機能

  • 販売店側：申込・登録、商品登録管理、在庫管理、キャンペン作成、売上管理、問合せ・連絡
  • 一般ユーザ側：登録、商品検索、注文、お支払、評価、コメント、問い合せ、商品注文履歴
  • システム管理：販売店管理、ユーザ管理、決済管理、広告

アーキテクチャー

マイクロサービスデザインを活用し、システムのアーキテクチャーを構成しました。

フロントエンド
HTTPSよりバックエンドが提供するREST APIを利用し、ユーザーなどと直接やり取りをする要素のことを構成します。

Webアプリ
React.JS, Bootstrap, CSSX, レスポンシブ対応

iOSアプリ

API ゲートウェイ
多数のAPIをサーバサイドでモニタリング、認証、管理、セキュリティ設定
開発言語 : Java
フレームワーク: Spring Cloud API Gateway

バックエンド
処理を行って結果を出力したり、記録媒体に保存したりして、REST APIを提供します。
マイクロサービスデザインを活用して、サービスを多数のマイクロサービスを分けます。マイクロサービス内の連携はAMQPメッセジング（RabbitMQ）を利用します。

開発言語 : Java
フレームワーク: Spring Boot, Spring Cloud, Spring Data
